# AI Tools Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard designed to analyze the AI tools ecosystem. The dashboard provides insights into traffic trends, funding performance, category distribution, company growth, and global adoption patterns.

## Key Insights

- Productivity and AI Assistant categories attract the highest traffic.
- Funding distribution is highly concentrated among leading AI companies.
- Traffic patterns vary significantly across regions and categories.
- Strong relationships can be observed between funding and user adoption for several AI tools.

## Project Objective

The objective of this project is to provide a comprehensive view of the AI tools market by combining traffic, funding, category, and regional data into a single interactive dashboard.
